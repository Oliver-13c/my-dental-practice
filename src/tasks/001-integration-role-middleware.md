# Task 001: Integrate and Test Role Enforcement Middleware for API Routes

## Description
Integrate the role enforcement middleware into existing and new Next.js API routes. Ensure access control matches user roles as defined. Test all authentication flows including session expiration and refresh.

## Acceptance Criteria
- Middleware imported and applied in all API routes requiring auth.
- Unit and integration tests verify correct role gating.
- Unauthorized access attempts return 401 or 403 appropriately.
